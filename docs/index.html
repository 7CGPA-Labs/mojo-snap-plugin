Mojo Snap Console

Retro game emulation for Jellyfin media servers.

A high-performance Jellyfin plugin running vanilla RetroArch WebAssembly cores directly in your browser. Browse your ROM library, select a game, and play instantly — with full gamepad support, save state syncing, and zero configuration overhead.

Core Features

⚡

WebAssembly Powered

Runs fceumm, snes9x, genesis_plus_gx, gambatte, mgba, and ecwolf cores locally inside client browsers with native speed.

🎮

Universal Gamepads

Auto-detects USB and Bluetooth controllers via the standard W3C Gamepad API.

📱

Smart TV Virtual Controller Support

Features a Dual TCP/UDP Architecture that allows mobile app gamepad inputs via mDNS and WebSocket relay for playing directly on Smart TVs.

💾

Server-Synced Saves

Save states are stored on your Jellyfin server, keeping your progress portable across all your devices.

⚙️

EmulatorJS-Style Controls

Remap gamepad buttons, load cheats, adjust video/audio settings, and manage save states from an overlay toolbar.

Quick Installation

Jellyfin Plugin

  1. Download the latest MojoSnapPlugin.dll from the Releases page.
  2. Copy the DLL into your Jellyfin server's plugins/ folder.
  3. Restart your Jellyfin server.
  4. Add your ROM folder as a library and play games natively from the web UI.
